هدف این دیزاین پترن Encapsulate متد هایی که بر روی یک object پردازش و یا عملیات انجام میدهند و سپس ان object را تغییر میدهند و یا manipulate میکنند در class های مجزا میباشد !!!

این Design Pattern در جایی استفاده میشود که یک class دارای method هایی باشد که آن method ها بر روی object هایش یک پردازش و یا عملیات انجام میدهند و سپس ان object را تغییر میدهند و یا آن را manipulate میکنند !!!
